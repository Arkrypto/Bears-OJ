# InnoSetup

> 打包可执行EXE文件和运行环境为安装包

## 一、准备

用`Exe4j`打包成的`exe`文件需要相应的`JDK`环境才能正常运行，为了解决这一问题，使用`InnoSetup`将`exe`和`jre`打包成一个完整的安装包

将`JDK`中的`jre`文件夹拷出，将`jre`和要打包的`exe`放置在同一文件夹

## 二、使用

点击右上角`file`选择`new`一个应用，依次填写应用名称、版本、出版者、官网

一直点击下一步直到选择`exe`应用的界面

1. 在`Application main executable file`处选择你要打包的`exe`文件
2. 添加环境文件夹，即包含了`jre`和`exe`的文件夹

如第一步选择`/user/admin/home/demo/demo.exe`，那么第二步则选择`/user/admin/home/demo/*`，其中`jre`包含在`/demo/`目录下

无脑下一步，直到设置输出目录以及安装包程序名称。继续下一步，使用默认脚本构建安装包程序，等待即可

## 三、善后

日常使用的`JDK`的`jre`略大，会造成最终程序较大，至少`>200MB`，可在网上寻找精简版的`jre`进行打包，效果一样
